About Our Project

  • What is this project?
    This is a Smart Attendance System that uses face recognition technology to automatically mark and record student attendance.
  • Team Information:
    Project Leader: Shrinidhi Haribhattanavar
    Team Members: Shreeram C Machak, D N Akshay, Nikhil Salunke
    Institution: Vishwanathrao Deshpande Institute of Technology, Haliyal
  • Technology Stack:
    Python (Face Recognition), Flask, MongoDB, Bootstrap, JavaScript, OpenCV
  • Objective:
    To reduce manual workload, prevent proxy attendance, and streamline the attendance tracking process with real-time face detection.
